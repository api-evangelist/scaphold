# ScapholdThis is a repo for managing machine readable artifacts for the [Scaphold APIs](https://scaphold.io). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Scaphold APIs](https://scaphold.io).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://scaphold.io)- [Developer](https://scaphold.io)- [Twitter](https://twitter.com/scapholddotio)- [GitHub](https://github.com/scaphold-io)The [APIs.json](https://github.com/api-evangelist/scaphold/blob/master/apis.json) index provides links to all the relevant links for the Scaphold APIs, including the OpenAPI and Postman if they are present.
